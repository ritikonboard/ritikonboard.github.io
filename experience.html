Research & Experience

My work focuses on embedded systems, electronics, video encoding, computer vision, and autonomous vehicles. I'm particularly interested in the intersection of hardware design and intelligent systems.

WISE Lab, IIIT Delhi

Research Intern • June 2024 - Present

Working under the guidance of Dr. Arani Bhattacharya on 'Cloud-Assisted Autonomous Driving over Wireless Network'. The main goal is to achieve minimum network latency in real-time video streaming of vehicle perception data and enable teleoperation over cloud networks.

Key Contributions:

  • Integrated CARLA simulator with FFmpeg for efficient encoding and streaming of video data over RTSP streams, optimizing parameters to minimize latency
  • Developed teleoperation/manual control features on cloud server using keyboard input for real-time vehicle control
  • Tested the complete setup across various video resolutions and network conditions, achieving significant latency improvements
  • Modified PYLOT's Erdos pipeline to enable cloud-based object detection using YOLOv8, improving accuracy in complex scenarios
  • Implemented adaptive model switching between server (YOLOv8x) and vehicle (YOLOv8n) based on network conditions

Presented poster titled "Cloud-Assisted Autonomous Driving Over Wireless Network" at ACM COMPASS 2024 held at IIIT Delhi, July 8-11, 2024.

Current work involves enhancing the autonomous vehicle perception capabilities.
